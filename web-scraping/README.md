## web_scraping_venv
- .gitignore the venv.
- Add a requirements.txt file for all used libraries
- Never move a venv after creation. It messes with the paths.

```bash
python -m venv web_scraping_venv

# Installs
pip install numpy
pip install beautifulsoup4
```
